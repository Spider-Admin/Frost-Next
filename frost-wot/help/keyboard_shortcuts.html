Keyboard Shortcuts (Frost-Next)

Back to Help Index

# Global Shortcuts

  • Ctrl + F = Open a Universal Search popup in most tables (such as message lists and file transfers), as well as in the text content of a message. See the "Universal Search" feature detail page for more information on how the feature works.
  • Ctrl + Shift + F = Open a case-sensitive Universal Search popup instead.

# Universal Search

  • Down Arrow or Ctrl + G = Go to the next result.
  • Up Arrow or Ctrl + Shift + G = Go to the previous result.
  • Escape = Close the search popup (can also be done by clicking outside of it).

# Board Message List

  • Left Arrow (in threaded mode) = Collapse the current message thread.
  • Right Arrow (in threaded mode) = Expand the current message thread.
  • F5 = Refresh the current board (download new messages).
  • Delete or Backspace = Remove the selected messages.
  • Shift + Delete or Shift + Backspace = Undelete the selected messages.
  • Enter = Opens the currently selected message in an independent message viewer window.
  • 1 or Numpad 1 = Mark selected identities as FRIEND.
  • 2 or Numpad 2 = Mark selected identities as GOOD.
  • 3 or Numpad 3 = Mark selected identities as NEUTRAL.
  • 4 or Numpad 4 = Mark selected identities as BAD.
  • Shift + T = Toggle "Display conversations as threads".
  • N = Go to the next unread message.
  • Shift + N = Toggle "Show only unread messages".
  • F = Mark the selected messages as FLAGGED.
  • Shift + F = Toggle "Show only flagged messages".
  • S = Mark the selected messages as STARRED.
  • Shift + S = Toggle "Show only starred messages".
  • J = Mark the selected messages as JUNK.
  • Shift + J = Toggle "Show JUNK messages too (otherwise they are hidden)".

# New Message Composer Window

  • Ctrl + Z = Undo. (Note: Be aware that changing your From identity in the message composer clears your current undo-history, to prevent faked identities.)
  • Ctrl + Shift + Z or Ctrl + Y = Redo.
  • Ctrl + W (or Cmd + W on Mac) = Close and abort the current message composer without sending the message.

# Independent Message Viewer Window

This is the window that appears if you double-click on a message or use Enter in various message lists. It displays a single message in a dedicated window.

  • Escape or Ctrl + W (or Cmd + W on Mac) = Close the current message viewer window.

# Message Search Window

  • Enter = If the message result list has keyboard focus, Enter opens the currently selected message in an independent message viewer window. Otherwise, Enter starts/stops the current search.

# Manage Downloaded Keys

  • Delete = Remove the selected keys.
  • Shift + Delete = Remove all keys from the same board as the selected key.
  • Shift + R = Refresh the table to display the most up-to-date list of keys.
  • D = Download the selected keys again.

# Manage Identities (aka Identities Browser)

  • 1 or Numpad 1 = Mark selected identities as FRIEND.
  • 2 or Numpad 2 = Mark selected identities as GOOD.
  • 3 or Numpad 3 = Mark selected identities as NEUTRAL.
  • 4 or Numpad 4 = Mark selected identities as BAD.

# Own Signature Editor

  • Ctrl + Z = Undo.
  • Ctrl + Shift + Z or Ctrl + Y = Redo.

# Key Conversion Utility

  • Ctrl + Z = Undo.
  • Ctrl + Shift + Z or Ctrl + Y = Redo.

# Downloads Tab

  • Delete = Remove the selected files from the queue.
  • 0 to 6 or Numpad 0 to Numpad 6 = Set the priority of the selected files (anywhere from 0 aka highest, to 6 aka paused).
  • Enter = Open the selected file with its associated application (if download is finished).
  • Shift + D = Open the selected download directories.
  • Shift + S = Force the selected files to start downloading immediately, if they're currently waiting.
  • Shift + E = Invert the "Enabled" state of the selected files.

# Uploads Tab

  • Delete = Remove the selected files from the queue.
  • 0 to 6 or Numpad 0 to Numpad 6 = Set the priority of the selected files (anywhere from 0 aka highest, to 6 aka paused).
  • Enter = Open the selected file with its associated application.
  • Shift + D = Open the selected upload directories.
  • Shift + S = Force the selected files to start uploading immediately, if they're currently waiting.
  • Shift + E = Invert the "Enabled" state of the selected files.

# "Add New Downloads" Window

  • Delete = Remove the selected files from the list.
  • Shift + Delete = Remove everything except the selected files from the list.
  • 0 to 6 or Numpad 0 to Numpad 6 = Set the priority of the selected files (anywhere from 0 aka highest, to 6 aka paused).
  • E = Invert the "Enabled" state of the selected files.
  • R = Rename the selected files (allows you to change the filename of the download).
  • P = Set a custom filename prefix for the selected files.
  • D = Change the download directory of the selected files.
  • V = Paste more keys from clipboard (only available if the "Add New Downloads" window was opened via the "Download keys from clipboard" toolbar button on the Downloads tab)
  • Ctrl + Enter = Okay! Add the files to your download queue.
  • Ctrl + Escape = Cancel! Close the window without queueing the files.

# "Add New Uploads" Window

  • Delete = Remove the selected files from the list.
  • Shift + Delete = Remove everything except the selected files from the list.
  • 0 to 6 or Numpad 0 to Numpad 6 = Set the priority of the selected files (anywhere from 0 aka highest, to 6 aka paused).
  • E = Invert the "Enabled" state of the selected files.
  • R = Rename the selected files (allows you to change the filename that the upload will be labeled with).
  • P = Set a custom filename prefix for the selected files.
  • D = Invert the "Compression" state of the selected files.
  • S = Set the custom crypto key for the selected files.
  • G = Generate a random crypto key for the selected files.
  • C = Copy the current crypto key(s) used for the selected files (to your clipboard).
  • Z = Clear the custom crypto key for the selected files.
  • A = Add more files to the list.
  • Ctrl + Enter = Okay! Add the files to your upload queue.
  • Ctrl + Escape = Cancel! Close the window without queueing the files.
